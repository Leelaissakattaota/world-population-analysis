## 📌 Project Overview
Scrapes the **live world population table** from Worldometers.info using BeautifulSoup and requests, then structures it into a Pandas DataFrame for analysis. Extracts all table headers and rows to produce a clean, structured dataset of world population figures by country for 2024.

**Domain:** Web Scraping — Global Demographics  
**Source:** worldometers.info/world-population  
**Libraries:** BeautifulSoup · requests · Pandas · lxml  

## 🚀 Pipeline
```
requests.get(Worldometers URL)
  → BeautifulSoup HTML parse
  → Extract <table> headers + rows
  → pandas.DataFrame → Analysis
```
